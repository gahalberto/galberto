# Imagens de Imóveis

Esta pasta pode ser usada para armazenar imagens de imóveis localmente.

**Nota:** Atualmente o sistema usa URLs de imagens externas. Para usar imagens locais, você pode:
1. Fazer upload das imagens aqui
2. Usar a URL: `/images/imoveis/nome-da-imagem.jpg` ao cadastrar o imóvel
